# windows安装步骤
### 推荐使用phpStudy集成环境
### 如果你不用集成环境，请自身确认开启了 `php-pdo-sqlite` 和 `php-sqlite3` 扩展

    phpStudy集成环境下载安装地址： https://www.xp.cn/ 安装后

    **请点击“WNMP”启动**
    
    点击左侧菜单“网站”，然后点击右边按钮“管理”-“php扩展”，确保”php-pdo-sqlite”和”php-sqlite3”开启

    如果它没开启，你可以点击它开启。不懂的话，把所有含有sqlite关键字的扩展都开启就好了

    网站对应目录就是该项目
